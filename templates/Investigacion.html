Facebook

Se lanzó en 2004 y su país de origen es Estados Unidos. Para el backend usan PHP y luego cambiaron a Hack. Para el frontend utilizan React (biblioteca de JavaScript creada por Facebook). Otros frameworks y bibliotecas son: React Native (para las versiones móviles) y GraphQL (para búsquedas más eficientes).

WhatsApp

Se lanzó en 2009 y su país de origen es Estados Unidos. Utilizan Erlang (para manejar la gran cantidad de usuarios), FreeBSD, Yaws (para los servidores web), Mnesia (para la base de datos), y para la encriptación de mensajes utilizan Signal.

Instagram

Se lanzó en 2010 y su país de origen es Estados Unidos. Utilizan Python para el backend, Django como framework web y PostgreSQL para la base de datos. Para las versiones móviles usan React Native.

Spotify

Se lanzó en 2006 y su país de origen es Suecia. Utilizan Python, Java, Hadoop, Apache Storm (para el procesamiento en tiempo real), Kafka y Google Cloud para su infraestructura.

TikTok

Se lanzó en 2016 y su país de origen es China. Utilizan Python, Go y C++ para el backend, TensorFlow para tecnologías de Machine Learning, además de Oracle Cloud y AWS para su infraestructura en EE. UU.
